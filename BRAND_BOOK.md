## Brand Overview

### Mission
Altyr is an exclusive content platform that finally feels pleasurable. We combine modern UX, extensive business tooling, and rewarding, gamified fan experiences to systematically grow creator revenue.

### Core Values
- **Creator-First**: Built with creators, not just for them
- **Pleasurable Experience**: Every interaction should feel delightful, not transactional
- **Data-Driven**: Leverage proven gamification techniques from gaming, social media, and e-commerce
- **Premium Quality**: Modern, responsive, and thoughtfully designed
- **Transparency**: Clear, direct communication with creators and fans

### Brand Personality
- **Modern & Edgy**: Forward-thinking, not traditional
- **Premium**: High-quality, sophisticated, but approachable
- **Confident**: Direct, clear, no fluff
- **Engaging**: Interactive, gamified, rewarding
- **Creator-Centric**: Empowering, supportive, partnership-focused

## Brand Voice & Tone

### Voice Characteristics

**Direct & Clear**
- Use straightforward language
- Avoid jargon unless necessary
- Be specific, not vague

**Confident but Approachable**
- Assertive without being arrogant
- Professional but not corporate
- Bold statements backed by substance

**Creator-Focused**
- Speak to creators as partners, not customers
- Acknowledge their expertise and needs
- Emphasize empowerment and growth

**Modern & Fresh**
- Use contemporary language
- Avoid clichés and overused phrases
- Feel current, not trendy

### Tone Variations

**Hero/Marketing Copy (Investor Site)**
- Bold, aspirational
- Emphasize transformation and opportunity
- Example: "an exclusive content platform that actually feels pleasurable"

**Hero/Marketing Copy (Waitlist Site)**
- Inviting, forward-looking
- Emphasize next generation and exclusivity
- Example: "The Next Generation of Exclusive Content"
- Tagline: "Built for creators. Designed for fans."

**Product Features**
- Clear, benefit-focused
- Highlight practical advantages
- Example: "Modern and responsive UX • Extensive Business tooling • Rewarding, gamified fan experiences"

**Investor/Professional**
- Data-driven, confident
- Focus on market opportunity and execution
- Example: "Subscription and adult content represent a massive, high-margin market"

**User-Facing (Waitlist)**
- Friendly, supportive
- Clear instructions and feedback
- Example: "You're on the list! We'll be in touch soon."
- Example: "Join the Waitlist" or "Be the first to know when we launch."

**Problem Statement (Waitlist)**
- Direct, honest about industry issues
- Emphasize what's wrong, then what's right
- Example: "The industry leader stopped innovating."
- Example: "Creators deserve modern tools, faster payouts, and real transparency."

**Value Proposition (Waitlist)**
- Emphasize premium quality and next generation
- Example: "ALTYR is the premium platform built for the next generation."
- Example: "Designed from the ground up with creators and fans at the center of everything."

### Do's and Don'ts

**Do:**
- Use "pleasurable" to describe the experience
- Emphasize "actually" when highlighting differentiation
- Use active voice
- Be specific about benefits
- Use contractions for conversational tone

**Don't:**
- Use corporate buzzwords
- Overpromise or exaggerate
- Use passive voice unnecessarily
- Be vague about features
- Sound like every other platform

## Typography

### Font Stack

**Primary Font Family**
```
system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif
```

**Rationale**
- Native system fonts for performance
- Consistent across platforms
- Modern, clean appearance
- No external font loading required

### Font Weights

**Extralight (200)**
- Usage: Large headlines, hero text
- Creates elegant, premium feel
- Example: "an exclusive content platform that actually feels pleasurable"

**Light (300)**
- Usage: Body text, descriptions, feature lists
- Maintains readability with sophistication
- Example: "Modern and responsive UX"

**Medium (500)**
- Usage: Emphasized text, important numbers
- Example: "$1,200,000" in fundraising callouts

**Semibold (600)**
- Usage: Logo wordmark, major headings
- Example: "ALTYR" logo

### Letter Spacing

**Tight (`tracking-tight`)**
- Usage: Headlines, hero text
- Creates modern, compact feel

**Normal (default)**
- Usage: Body text, descriptions
- Optimal readability

**Wide (`tracking-wide`)**
- Usage: Feature bullets, labels
- Adds emphasis and clarity

**Extra Wide (`tracking-[0.12em]`)**
- Usage: Logo wordmark
- Creates distinctive, premium look

**Ultra Wide (`tracking-[0.35em]`)**
- Usage: Section labels, small caps
- Example: "MARKET OPPORTUNITY"

### Font Sizes

**Hero/Display**
- `text-8xl` (96px) - Large hero logos
- `text-7xl` (72px) - Hero headlines
- `text-6xl` (60px) - Major section headings
- `text-5xl` (48px) - Large headings

**Headings**
- `text-4xl` (36px) - Section titles
- `text-3xl` (30px) - Subsection titles
- `text-2xl` (24px) - Card titles
- `text-xl` (20px) - Feature titles

**Body**
- `text-lg` (18px) - Large body text, descriptions
- `text-base` (16px) - Standard body text
- `text-sm` (14px) - Supporting text
- `text-xs` (12px) - Labels, metadata

### Line Height

**Tight (`leading-[1.3]`)**
- Usage: Headlines, short phrases
- Creates compact, modern look

**Normal (default ~1.5)**
- Usage: Body text
- Optimal readability

**Relaxed (`leading-relaxed`)**
- Usage: Long-form content, descriptions
- Improves readability for extended text

## Design Principles

### 1. Dark-First Design
- Always start with dark purple background (`#18021A`)
- Use light text on dark backgrounds
- Create depth through layering and gradients

### 2. Glassmorphism
- Use subtle backdrop blur effects
- Semi-transparent overlays (2-6% opacity)
- Border definitions with low opacity (8-12%)
- Creates modern, premium aesthetic

### 3. Gradient Accents
- Use gradients sparingly but strategically
- Primary: Magenta to Purple (`#AC0064` → `#64109A`)
- Logo: Purple to Orange (`#9B4DCA` → `#E85A24` → `#FF8C42`)
- Apply to text, backgrounds, and borders

### 4. Subtle Motion
- All animations should feel natural and purposeful
- Use easing functions: `cubic-bezier(0.25, 0.46, 0.45, 0.94)`
- Duration: 0.2s (micro), 0.5-0.8s (standard), 1s+ (hero)
- Never distract from content

### 5. Spacious Layout
- Generous padding and margins
- Section spacing: `py-24 md:py-32` (96-128px)
- Content max-width: `max-w-5xl` to `max-w-6xl`
- Center-aligned content for hero sections

### 6. Rounded Corners
- Small: `rounded-xl` (12px) - Buttons, inputs
- Medium: `rounded-2xl` (16px) - Cards, containers
- Large: `rounded-3xl` (24px) - Feature cards, major sections
- Full: `rounded-full` - Pills, badges, circular elements

## Animation & Motion

### Animation Principles

**Purposeful Motion**
- Every animation should serve a purpose
- Enhance understanding, not distract
- Respect user preferences (prefers-reduced-motion)

**Natural Easing**
- Primary easing: `cubic-bezier(0.25, 0.46, 0.45, 0.94)`
- Smooth, organic feel
- Never use linear animations

**Staggered Animations**
- Delay: `index * 0.1` to `index * 0.15`
- Creates cascading, polished effect
- Use for lists, cards, features

### Common Animations

**Fade In**
- `opacity: 0` → `opacity: 1`
- Duration: 0.6-1s
- With slight Y translation: `y: 20` → `y: 0`

**Scale on Hover**
- `scale: 1` → `scale: 1.02`
- Duration: 0.2s
- Subtle, not dramatic

**Rotate on Hover**
- Small rotations: `rotate: 3deg` to `rotate: 6deg`
- Duration: 0.3s
- Adds playfulness

**Pulse/Glow**
- Opacity: `[0.3, 0.5, 0.3]`
- Scale: `[1, 1.1, 1]`
- Duration: 4s, infinite
- Creates atmospheric effect

### Scroll Animations

**Viewport-Based**
- Trigger: `whileInView`
- Once: `viewport={{ once: true }}`
- Margin: `margin: "-60px"` to `margin: "-80px"`
- Creates reveal-on-scroll effect

## Usage Guidelines

### Do's

✅ **Use dark purple as primary background**
✅ **Apply gradients sparingly and strategically**
✅ **Maintain consistent spacing and padding**
✅ **Use system fonts for performance**
✅ **Apply subtle animations with natural easing**
✅ **Use opacity variations for text hierarchy**
✅ **Maintain glassmorphism aesthetic**
✅ **Center-align hero content**
✅ **Use rounded corners consistently**
✅ **Test on mobile devices**

### Don'ts

❌ **Don't use pure white backgrounds**
❌ **Don't overuse gradients**
❌ **Don't use harsh animations**
❌ **Don't use too many font weights**
❌ **Don't break the dark theme**
❌ **Don't use low contrast text**
❌ **Don't ignore mobile responsiveness**
❌ **Don't use sharp corners**
❌ **Don't clutter with too many effects**
❌ **Don't deviate from the color palette**

### Accessibility

**Contrast Ratios**
- Ensure WCAG AA compliance (4.5:1 for text)
- Test all text on dark backgrounds
- Use opacity to create hierarchy, not just color

**Motion**
- Respect `prefers-reduced-motion`
- Provide alternative static states
- Don't rely solely on animation for information

**Touch Targets**
- Minimum 44x44px for interactive elements
- Adequate spacing between clickable items
- Clear hover and active states

- Lucide React (icons)

**Styling Approach**
- Utility-first CSS (Tailwind)
- Custom color values via arbitrary values
- CSS-in-JS for complex animations
- Responsive design with mobile-first approach

### Code Patterns

**Color Usage**
```jsx
// Background
className="bg-[#18021A]"

// Text with opacity
className="text-white/90"

// Gradients
className="bg-gradient-to-r from-[#AC0064] to-[#64109A]"

// Borders with opacity
className="border border-white/[0.08]"
```

**Animation Pattern**
```jsx
<motion.div
  initial={{ opacity: 0, y: 20 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ 
    duration: 0.8, 
    ease: [0.25, 0.46, 0.45, 0.94] 
  }}
>
```

whileHover={{
