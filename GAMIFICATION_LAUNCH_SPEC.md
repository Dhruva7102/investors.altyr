# Altyr Gamification Launch Spec
**Version 1.0 - Launch Features**

## Philosophy
Introduce gamification systems that feel natural and familiar (like Discord, gaming platforms) without overwhelming users switching from existing platforms. Set foundation for future expansion.

---

## 1. Global XP & Leveling System

### Core Mechanics

**XP Sources (Weighted)**
- **Daily Login**: +50 XP (once per day)
- **View Content**: +5 XP per post/video viewed (max 10/day = 50 XP)
- **Send Message**: +10 XP per message sent (max 5/day = 50 XP)
- **Tip/Custom Request**: +25 XP per tip
- **Subscribe**: +100 XP (one-time per creator)
- **Renew Subscription**: +50 XP per renewal
- **Purchase Content**: +15 XP per purchase
- **Leave Comment**: +5 XP per comment (max 3/day = 15 XP)

**Level Progression**
- **Level 1-10**: 100 XP per level (quick early wins)
- **Level 11-25**: 200 XP per level
- **Level 26-50**: 300 XP per level
- **Level 51-75**: 400 XP per level
- **Level 76-100**: 500 XP per level
- **Level 100+**: 600 XP per level (infinite progression)

**Visual Display**
- Level badge next to username (everywhere)
- XP progress bar (subtle, in profile)
- Level-up animation (celebratory but not intrusive)
- "Level X" text with optional level name (e.g., "Level 15 - Devoted Fan")

### Level Rewards (Unlock System)
- **Level 5**: Unlock profile customization
- **Level 10**: Unlock first badge slot
- **Level 15**: Unlock second badge slot
- **Level 20**: Unlock profile banner customization
- **Level 25**: Unlock third badge slot
- **Level 30**: Unlock animated profile effects
- **Level 50**: Unlock exclusive profile theme
- **Level 100**: Unlock legendary status + special badge

## 2. Profile Customization System

### Profile Elements (Discord-Inspired)

**Avatar Frame/Border**
- Default: Simple circle/square
- Level 10+: Unlock colored borders (match brand colors)
- Level 20+: Unlock gradient borders
- Level 30+: Unlock animated borders (subtle pulse/glow)
- Level 50+: Unlock premium frames (unique designs)

**Profile Banner**
- Default: Solid color or gradient
- Level 20+: Unlock custom banner upload
- Level 30+: Unlock animated banners (subtle motion)
- Level 50+: Unlock premium banner templates

**Profile Theme/Color Scheme**
- Default: Platform dark theme
- Level 5+: Unlock accent color customization (choose from brand palette)
- Level 15+: Unlock custom color picker
- Level 30+: Unlock theme presets (Creator's Choice, Fan Favorite, etc.)
- Level 50+: Unlock exclusive themes (limited edition)

**Username Display**
- Default: Standard text
- Level 10+: Unlock gradient text option
- Level 25+: Unlock glow effect on username
- Level 50+: Unlock animated username effects

**Profile Badge Slots**
- 3 badge slots (unlocked at levels 10, 15, 25)
- Badges display prominently on profile
- Click to see badge details/earn date

**Status Indicator**
- Online/Offline status (like Discord)
- Optional: "Active X minutes ago"
- Optional: Custom status message (Level 20+)

### Customization UI
- Simple, intuitive editor (like Discord profile customization)
- Live preview
- Save/Discard changes
- "Default" reset option

**Engagement Badges**

- **Binge Watcher** (Silver): View 1,000 posts

- **Philanthropist** (Platinum): Spend $10,000 total

**Subscription Badges**

- **Devoted** (Silver): Subscribe to 1 creator for 6 months
- **Dedicated** (Gold): Subscribe to 1 creator for 12 months
- **Multi-Fan** (Bronze): Subscribe to 5 creators simultaneously
- **Super Fan** (Silver): Subscribe to 10 creators simultaneously
- **Ultra Fan** (Gold): Subscribe to 20 creators simultaneously

- **Early Adopter**: Join in first 30 days
- **Founding Member**: Join in first week
- **Beta Tester**: Join during beta (if applicable)

- **Level 100**: Reach level 100
- **Centurion**: Reach level 100 (special variant)

- **Holiday 2025**: Participate in holiday event

### Badge Design
- **Rarity Tiers**: Bronze (common), Silver (uncommon), Gold (rare), Platinum (epic), Legendary (mythic)

- **Tooltip**: Hover to see badge name, description, earn date

### Badge Display
- **Profile**: 3 slots (unlocked by level)
- **Comments/Messages**: Small badge icon next to username
- **Leaderboards**: Badge icons in rankings
- **Badge Collection**: Full gallery in profile

## 4. Daily Login Rewards

### Reward Structure

**Daily Streak System**
- Day 1: 50 XP + "Welcome Back" notification
- Day 2: 75 XP
- Day 3: 100 XP
- Day 4: 125 XP
- Day 5: 150 XP + Small bonus (profile accent color unlock)
- Day 6: 175 XP
- Day 7: 200 XP + **Weekly Bonus** (badge or customization item)
- Day 8+: 200 XP + streak multiplier (1.1x, 1.2x, etc. up to 2x at 30 days)

**Streak Protection**
- **Grace Period**: 24-hour window to maintain streak
- **Streak Freeze**: One "freeze" per month (prevent loss)
- **Visual Indicator**: Streak counter in profile/header

**Monthly Milestones**
- 7 days: "Week Warrior" badge
- 14 days: "Fortnight Fan" badge
- 30 days: "Monthly Master" badge + exclusive reward

### UI/UX
- **Login Modal**: Quick, non-intrusive popup showing reward
- **Progress Indicator**: Visual streak calendar
- **Notification**: Subtle notification when reward claimed
- **Skip Option**: Dismiss without claiming (still counts)

## 5. Meaningful Interaction Tracking

### Interaction Types

**Content Engagement**
- View post/video (5 XP, max 10/day)
- Like/React (3 XP, max 20/day)
- Comment (5 XP, max 3/day)
- Share (10 XP, max 2/day)

**Communication**
- Send message (10 XP, max 5/day)
- Reply to message (5 XP, max 10/day)
- Send tip (25 XP, unlimited)
- Custom request (30 XP, unlimited)

**Financial**
- Subscribe (100 XP, one-time per creator)
- Renew subscription (50 XP)
- Purchase content (15 XP, unlimited)
- Tip (25 XP, unlimited)

### Anti-Gaming Measures
- Daily caps on low-value actions (views, likes)
- No caps on high-value actions (spending, subscriptions)
- Cooldown periods for spam prevention
- Quality over quantity (meaningful interactions weighted higher)

## 9. Technical Considerations

### Database Schema (Simplified)
```sql
users:
  - id
  - xp (total)
  - level
  - daily_login_streak
  - last_login_date
  - profile_customization (JSON)

badges:
  - id
  - name
  - description
  - rarity
  - icon_url
  - category

user_badges:
  - user_id
  - badge_id
  - earned_date
  - display_order

xp_log:
  - user_id
  - xp_amount
  - source
  - timestamp
```

### Performance
- Cache level calculations
- Batch XP updates (not real-time)
- Optimize badge queries
- Lazy load customization assets

## 10. User Onboarding

### First-Time Experience

2. **Tutorial**: Quick 30-second overview of XP system
3. **First Action**: Guide user to view content → "You earned 5 XP!"
4. **Level Up**: Celebrate first level up (Level 2)
5. **Badge Unlock**: Show first badge earned
6. **Profile Customization**: Unlock at Level 5 with tutorial

### Help & Documentation
- **Tooltips**: Inline help throughout
- **FAQ**: "How do I earn XP?" section
- **Badge Gallery**: See all available badges
- **Progress Tracker**: "You're X XP away from Level Y"

## Notes

- **Keep it Simple**: Launch with core systems, expand later
- **Familiar Patterns**: Use Discord/gaming platform conventions
- **Non-Intrusive**: Don't force gamification, make it optional
- **Value First**: Gamification enhances experience, doesn't replace value
- **Creator Focus**: Always benefit creators, not just fans
